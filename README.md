# rafiqsaad.github.io

Personal portfolio for Saad Rafiq — software engineer specializing in robotics, machine learning, and full-stack development.

**Live site:** https://srblank.github.io/rafiqsaad/

## Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Framer Motion
- Deployed via GitHub Actions → GitHub Pages

## Local dev

```bash
npm install
npm run dev
```
